Docker Installation

Steps

  • * sudo apt Update

  • * sudo apt-get install apt-transport-https ca-certificates curl software-properties-common

  • * curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

  • * sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu focal stable"

  • * sudo Apt Update

  • * sudo apt install docker-ce / sudo apt installl docker.io

  • * Check the Docker Version command as " docker --version "

  • * Another command as to create a file as "touch [filename] "

To Create a Docker Image using Putty

  • * First Open Virtualbox

  • * Start the Machine

  • * Login to Root user

  • * Verify the IP Address

  • * Copy that IP Address to putty

  • * And Paste that IP Address to Putty in Place of Hostname

  • * Then Click OK

  • * Asking Login Details

  • * Then After First Check the Docker installed or not!

  • * Command as "docker --version"

  • * Create a Folder

  • * Command as "mkdir [foldername] "

  • * Change into that folder directory

  • * Command as "cd [foldername] "

  • * Create some file like "index.html"

  • * Command as -----> echo "hai" > index.html

  • * Check the file was created or not

  • * Command as "ls"

  • * Then Create a Dockerfile

  • * Command as "vi Dockerfile"

  • * It will go to editor page

  • * Press "I"

  • * Then Type some Content Like FROM httpd:latest WORKDIR /usr/local/apache2/htdocs COPY ./index.html/ . CMD ["httpd","-D","FOREGROUND"]
  • * Then press ESC button

  • * Type colon(:)wq

  • * It will back to Terminal

  • * Then check the Dockerfile created or not

  • * Command as "ls"

  • * Then Build the Image

  • * Command as " docker build -t [some name] .

  • * Then check the image was created or not

  • * Command as "docker images"

  • * Then run the Container

  • * Command as "docker run -d --name [somename] -p 81:80[some port] [image name] :latest

  • * Then the Container running or not check it

  • * Command as "docker ps"

  • * If we want to Stop the Container

  • * Command as "docker stop [container ID] "

  • * If we want to start the Container

  • * Command as "docker start [container ID] "

How to Push the Image from Putty Terminal to Docker Hub

  • * First login to docker in putty

  • * Command as "docker login"

  • * For logout command in putty has "docker logout"

  • * Give the Credentials like [Username] and [Password]

  • * And Push the image to Docker Hub

  • * Command as "docker tag [image name] : latest [Username]/dockerhub : [give some name]"

  • * Next Command as

  • * "docker push [Username]/dockerhub:[Above given name]"

Simple Commands for putty terminal to Docker Hub

  • * docker tag [image name] : latest [Username]/dockerhub : [give some name]

  • * docker push [Username]/dockerhub:[Above given name]

Pull Commands from Dockerhub to Putty

  • * "docker pull [Username]/dockerhub:[given name]
